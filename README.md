## Description    
Web application for cinema. The application is designed to work with movies and sessions.  
## Used technologies
* JDK 1.8  
* Spring 4 Framework  
* Maven v3.2.2
* Jetty v.9.4.8  
* H2 database v1.4.196

## Build  

       $mvn clean install  

## Run  

       $mvn jetty:run  
       
       http://localhost:8080/
